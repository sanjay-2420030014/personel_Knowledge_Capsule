# 🧠 Personal Knowledge Capsule  
### A Simple Web-Based Knowledge Management System

---

## 📌 Overview
Personal Knowledge Capsule is a full-stack web application that allows users to store, organize, and manage personal knowledge efficiently.  
It provides a centralized platform to save important notes, ideas, and study materials in a structured way.

---

## 🚀 Features
- ➕ Add new knowledge entries  
- 📖 View all stored knowledge  
- 🗑️ Delete unwanted entries  
- 🔒 Lock important content (basic level)  
- ⚡ Fast and lightweight system  
- 🌐 Simple and responsive UI  

---

## 🏗️ System Architecture
The project follows a layered architecture:

- **Frontend** → HTML, CSS, JavaScript  
- **Backend** → Spring Boot (REST API)  
- **Service Layer** → Business logic  
- **Repository Layer** → JPA (Database interaction)  
- **Database** → MySQL  

---

## 🛠️ Technologies Used
- Java  
- Spring Boot  
- Spring Data JPA (Hibernate)  
- MySQL  
- HTML, CSS, JavaScript  
- Git & GitHub
